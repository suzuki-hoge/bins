# Git Status ( gst / gstf )

## gst

alias to ***g***it ***st***atus.

```
$ gst -h
bins 0.1.0

USAGE:
    git-status

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information
```

examples.

``` 
$ gst

## master...origin/master
MM Cargo.lock
M  Cargo.toml
```

## gstf

***g***it ***st***atus with ***f***ilter selection.

```
$ gstf
```

```
> 
+-------------------------------------------------------------------------------------------------+
| MM Cargo.lock                                                                                   |
| M  Cargo.toml                                                                                   |
|                                                                                                 |
|                                                                                                 |
|                                                                                                 |
+-------------------------------------------------------------------------------------------------+
```
