# Git Push ( gps )

## gps

alias to ***g***it ***p***u***s***h.

```
$ gps -h
bins 0.1.0

USAGE:
    git-push [FLAGS]

FLAGS:
    -f, --force      force update
    -h, --help       Prints help information
    -V, --version    Prints version information
```

examples.

``` 
$ gps    # git push origin {current_branch}
```
