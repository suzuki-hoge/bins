# Git Add ( gad / gadf )

## gad

alias to ***g***it ***ad***d.

```
$ gad -h
bins 0.1.0

USAGE:
    git-add

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information
```

examples.

``` 
$ gad
```
